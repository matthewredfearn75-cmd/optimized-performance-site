Executive Summary: Optimized Performance is a direct-to-consumer research peptide company. The e-commerce platform is built on Next.js 14 / React 18, deployed on Vercel, with MoonPay cryptocurrency payment processing (fiat → USDC on Polygon). The website, admin dashboard, product catalog, and payment rail are functional. Key remaining work includes environment configuration, supplier sourcing, MoonPay KYB verification, and legal review before the May 19 launch.

1. Platform Architecture

Tech Stack

  • Framework: Next.js 14.2 + React 18.3
  • Hosting: Vercel (auto-deploy from GitHub)
  • Payment: MoonPay (fiat → USDC/Polygon)

4. Payment Infrastructure

Payment Flow: Customer pays with credit/debit card → MoonPay converts fiat to USDC (stablecoin on Polygon blockchain) → ~4% processing fee → funds arrive in configured wallet. MoonPay handles KYC for customers automatically.

  • SDK: @moonpay/moonpay-react v1.10.6 — integrated as overlay widget in checkout
  • Webhook: /api/webhooks/moonpay — verifies HMAC signature, updates order to "completed", decrements inventory, sends confirmation email
  • Current state: Using test/placeholder API key — needs production key + KYB verification with MoonPay
  • Off-ramp: USDC → fiat conversion and bank withdrawal setup still needed
